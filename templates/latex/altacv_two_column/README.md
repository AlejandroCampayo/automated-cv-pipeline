# Altacv Two-Column Template

Files:
- cv_template.tex (main CV file)
- page1sidebar.tex (sidebar content)

Usage:
1) Copy this folder into outputs/job_fairs/<name>/.
2) Rename cv_template.tex to cv.tex.
3) Update cv.tex and page1sidebar.tex with content.
4) Build: latexmk -pdf cv.tex

Notes:
- This template matches the two-column job_fairs CV style.
- Ensure altacv.cls is present in the same folder.

# Single-Column Article Template

Files:
- cv_template.tex (main CV file)

Usage:
1) Copy this folder into outputs/job_offers/<name>/.
2) Rename cv_template.tex to cv.tex.
3) Update cv.tex with content.
4) Build: latexmk -pdf cv.tex

Notes:
- This template matches the single-column Enovos/Amaris style.

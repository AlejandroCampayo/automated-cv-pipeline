# Northwind Labs — Machine Learning Engineer (LLM Applications)

Date: 2026-06-01
Location: Berlin, Germany (hybrid, 2 days/week on-site)
Contract: Permanent, full-time

## Summary
Northwind Labs builds LLM-powered products for the customer-support space. We're
hiring an ML Engineer to own our retrieval and evaluation stack end to end — from
data pipelines through model serving to measuring real-world quality.

## Responsibilities
- Design and ship RAG pipelines over large, messy document collections.
- Build offline and online evaluation harnesses; define quality metrics.
- Improve answer accuracy and latency of production LLM services.
- Set up CI/CD for prompts and models; prevent regressions.
- Collaborate with product and support teams; translate needs into ML work.

## Required Qualifications
- 3+ years building production ML or LLM systems.
- Strong Python; experience with PyTorch or a modern ML stack.
- Hands-on RAG / vector search experience (e.g. pgvector, FAISS).
- Experience with evaluation, A/B testing, and metrics.
- Comfortable with Docker and cloud (AWS or GCP).
- Fluent English. German is a plus, not required.

## Nice to Have
- Experience with LangChain or similar orchestration frameworks.
- Familiarity with prompt-engineering best practices and guardrails.

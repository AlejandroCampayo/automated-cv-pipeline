# How to Apply — Northwind Labs, ML Engineer (LLM Applications)

## Recommendation: **Apply directly + a short warm cold-email**
Strong fit (88/100), direct posting, no recruiter gate. Apply through the portal,
then send a brief note to a hiring contact to stand out from the portal pile.

## Channel notes
- **Portal:** primary application. Attach the tailored CV from this folder.
- **Cold email:** only adds value if it's specific. Lead with one measured result
  that maps to their hardest problem (evaluation quality), not a generic intro.

## Finding a contact
- Look for the hiring manager / ML lead on LinkedIn ("Northwind Labs" + "ML").
- Guess-and-verify email pattern (`first.last@northwindlabs.com`) only if listed publicly; otherwise use the portal + LinkedIn message.
